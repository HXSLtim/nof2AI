## 📈 归档策略

### 自动归档条件
1. **临时文档**: 完成特定任务后7天自动归档
2. **过时修复**: 被新方案替代的修复文档
3. **版本迭代**: 重大版本更新前的文档

### 手动归档条件
1. **重复内容**: 与其他文档重复的内容
2. **无效信息**: 不再适用的配置或说明
3. **测试文件**: 临时测试和验证文件
