## 🔧 核心配置项

### 环境配置
```env
# 数据库配置
DATABASE_URL="./data/trading.db"

# OKX API配置
OKX_API_KEY="your_api_key"
OKX_SECRET_KEY="your_secret_key"
OKX_PASSPHRASE="your_passphrase"
OKX_SANDBOX="true"  # 测试网环境

# AI配置
OPENAI_API_KEY="your_openai_key"
AI_MODEL="gpt-4"

# 服务器配置
PORT=3000
NODE_ENV="development"
```

### 交易配置
- **杠杆设置**: 默认1-10倍杠杆
- **风险控制**: 单笔交易最大金额限制
- **止损设置**: 默认-8%止损线
- **止盈设置**: 根据持仓时间动态调整

### 系统配置
- **数据采集间隔**: 3分钟价格数据
- **AI决策频率**: 5分钟一次决策
- **历史数据保留**: 7天自动清理
- **日志级别**: INFO/WARN/ERROR

## 🚀 快速配置指南

### 1. 基础环境配置
1. 复制 `example.env.local` 为 `.env.local`
2. 填写必要的API密钥
3. 选择运行环境（测试网/生产网）

### 2. OKX交易所配置
1. 在OKX申请API密钥
2. 设置必要的权限（交易、读取）
3. 配置IP白名单（推荐）
4. 测试API连接

### 3. AI功能配置
1. 获取OpenAI API密钥
2. 选择合适的AI模型
3. 配置提示词模板
4. 测试AI决策功能
